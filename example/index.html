WeUI

WeUI 是一套同微信原生视觉体验一致的基础样式库，由微信官方设计团队为微信 Web 开发量身设计，可以令用户的使用感知更加统一。包含button、cell、dialog、 progress, toast、article、icon等各式元素。

WeUI 本身只包含样式，不包含 javascript 功能。此组件库基于zepto对WeUI做深度定制，以满足简单常用的开发需求。
